#Readme
How to make movie trailer websites!

To make you movie trailer website, edit the entertainment_center.py code.  

1. Replace you movies, trailers etc... and assigne them to variables named after the name of the movie.
2.  Change the names of the variables assigned to the movies variable with your new movie names, which you assigned the data.

3. Save and Run!

4.  The webpage should be opened and saved in the same folder under the name fresh_tomatoes.html
